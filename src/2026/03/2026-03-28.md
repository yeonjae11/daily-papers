### 1. EVA: Efficient Reinforcement Learning for End-to-End Video Agent

**트랙**: LLM Post-training | **링크**: [arXiv](https://arxiv.org/abs/2603.22918)

제공된 abstract와 메타데이터를 바탕으로 분석하겠습니다.

📋 Problem Definition

멀티모달 대형 언어모델(MLLM)을 활용한 비디오 이해는 비디오의 긴 토큰 시퀀스, 광범위한 시간적 의존성, 그리고 중복 프레임으로 인해 여전히 어려운 과제이다. 기존 접근법들은 MLLM을 수동적 인식기(passive recognizer)로 취급하여 전체 비디오 또는 균일하게 샘플링한 프레임을 처리하며, 질의에 맞춘 적응적 추론을 수행하지 못한다. 최근의 에이전트 기반 방법들도 외부 도구를 도입하지만, 수동 설계된 워크플로우와 인식 우선(perception-first) 전략에 의존하여 긴 비디오에서 비효율적이다.

📚 Background / Related Works

비디오 이해 분야에서 기존 MLLM 기반 접근법들은 VideoLLaMA, Video-ChatGPT 등과 같이 전체 프레임을 일괄 처리하는 방식이 주류였으며, 이는 긴 비디오에서 컨텍스트 윈도우 제한과 중복 정보 처리라는 근본적 한계를 갖는다. 적응적 에이전트 방식으로는 VideoAgent, VideoTree 등이 외부 도구(캡셔닝, 세그먼테이션 등)를 활용해 비디오를 탐색하는 전략을 제안했으나, 미리 정의된 파이프라인에 의존하여 유연성이 부족했다. 강화학습 기반 후처리 훈련 측면에서는 GRPO(Group Relative Policy Optimization)와 KTO(Kahneman-Tversky Optimization) 같은 최적화 기법이 LLM 정렬(alignment)에 성과를 보여왔으며, EVA는 이를 비디오 에이전트 훈련에 최초로 결합 적용한다.

🔬 Main Methodology

EVA의 핵심은 "planning-before-perception" 패러다임으로, 비디오를 먼저 전부 보는 것이 아니라 질의(query)를 기반으로 무엇을(what), 언제(when), 어떻게(how) 볼지를 자율적으로 결정하는 반복적 추론 루프를 구현한다. 이 루프는 Summary-Plan-Action-Reflection의 4단계로 구성되며, 에이전트가 현재까지의 정보를 요약하고, 다음 행동을 계획하며, 특정 구간의 프레임을 선택적으로 관찰하고, 결과를 반성하여 다음 반복으로 넘어가는 구조이다. 훈련은 3단계 파이프라인으로 설계되었다: (1) SFT(Supervised Fine-Tuning) 단계에서 에이전트의 기본 추론 형식과 도구 사용 패턴을 모방 학습하고, (2) KTO(Kahneman-Tversky Optimization) 단계에서 행동의 선호/비선호 쌍 없이도 바람직한 행동과 바람직하지 않은 행동을 구분하는 정렬 학습을 수행하며, (3) GRPO 단계에서 그룹 내 상대적 보상을 기반으로 정책을 최적화하여 최종 성능을 끌어올린다. 각 훈련 단계를 위한 고품질 데이터셋도 별도로 구축하여 안정적이고 재현 가능한 훈련을 지원한다.

🧪 Evaluation

EVA는 6개의 비디오 이해 벤치마크에서 평가되었으며, 일반 MLLM 베이스라인 대비 6-12%의 성능 향상을, 기존 적응적 에이전트 방법 대비 1-3%의 추가 성능 향상을 달성했다. SFT에서 KTO, GRPO로 이어지는 단계적 훈련이 supervised imitation과 강화학습 사이의 간극을 효과적으로 연결하며, 단순히 모든 프레임을 처리하는 방식보다 선택적 관찰 전략이 정확도와 효율성 모두에서 우수함을 실험적으로 입증했다. 코드와 모델이 공개되어 재현성을 확보한 점도 주목할 만하다.

💡 Key Intuition & Lesson

이 논문의 핵심 인사이트는 비디오 이해에서 "먼저 다 보고 판단하기"보다 "먼저 계획하고 필요한 부분만 보기"가 더 효과적이라는 것이며, 이는 인간이 긴 영상을 이해할 때 질문에 관련된 부분을 선택적으로 탐색하는 인지 전략과 일치한다. SFT 단독으로는 에이전트의 탐색 행동을 충분히 최적화할 수 없으며, KTO와 GRPO를 순차적으로 적용하는 3단계 커리큘럼이 모방 학습의 한계를 극복하는 데 필수적임을 보여준다. 이는 비디오 영역을 넘어, 긴 컨텍스트를 다루는 모든 MLLM 에이전트 설계에서 "능동적이고 질의 주도적인 정보 탐색"과 "단계적 강화학습 훈련"이라는 두 가지 설계 원칙의 중요성을 시사한다.

### 2. Autoregressive vs. Masked Diffusion Language Models: A Controlled Comparison

**트랙**: Diffusion Language Models | **링크**: [arXiv](https://arxiv.org/abs/2603.22075)

📋 Problem Definition

자기회귀(AR) 언어 모델과 마스크 확산(MDLM) 언어 모델은 각각 독립적으로 발전해왔지만, 기존 비교 연구들은 서로 다른 데이터셋, 모델 크기, 연산 예산을 사용하여 생성 패러다임 자체의 차이를 순수하게 분리하기 어려웠다. 본 논문은 데이터, 연산량, 하드웨어를 완전히 동일하게 통제한 환경에서 두 패러다임의 학습 역학, 수렴 양상, 생성 품질 차이를 실증적으로 규명하고자 한다.

📚 Background / Related Works

자기회귀 쪽에서는 GPT(Radford et al., 2019), LLaMA(Touvron et al., 2023)로 대표되는 좌-우 순차 생성 패러다임이 지배적이며, Kaplan et al.(2020)과 Hoffmann et al.(2022)의 스케일링 법칙 연구가 모델 크기-데이터-성능 관계를 정립했다. 확산 언어 모델 쪽에서는 D3PM(Austin et al., 2021)이 이산 확산을 도입하고, MDLM(Sahoo et al., 2024)이 마스킹 기반 프레임워크로 단순화했으며, SEDD(Lou et al., 2024)가 ICML 2024 Best Paper를 수상하고, LLaDA(Nie et al., 2025)가 8B 파라미터에서 LLaMA3과 경쟁적 성능을 달성했다. 그러나 기존 비교들은 모두 파라미터 수, 학습 데이터, 연산 예산이 달라 패러다임 자체의 효과를 분리할 수 없었다는 한계가 있다.

🔬 Main Methodology

TinyStories 50M 토큰 데이터셋에서 동일한 하이퍼파라미터(20,000 스텝, 배치 크기 32, 시퀀스 길이 512, d_model=768, 12 레이어, AdamW 옵티마이저)와 동일한 NVIDIA H100 GPU 위에서 AR 트랜스포머(123.6M 파라미터)와 MDLM 트랜스포머(162.7M 파라미터)를 학습시켰다. AR 모델은 인과적(causal) 어텐션 마스크로 다음 토큰 예측을 수행하고, MDLM은 양방향 어텐션과 코사인 노이즈 스케줄(gamma(t) = 1 - cos^2(pi*t/2))을 사용해 마스크된 토큰을 예측한다. MDLM의 파라미터가 31.6% 더 많은 것은 타임스텝 임베딩 모듈 등 패러다임 고유의 구조적 요구 때문이며, 인위적으로 줄이지 않았다. 생성 시 AR은 nucleus sampling(p=0.9, temperature 0.8)을, MDLM은 완전 마스크 시퀀스에서 100 스텝에 걸쳐 신뢰도 기반 언마스킹(temperature annealing 1.2→0.5, repetition penalty 1.3)을 적용했다. 평가는 1,000개 독립 샘플(각 128 토큰)에 대해 Distinct-n, Self-BLEU, 고유 첫 5-gram 비율 등 다양성 지표를 정량 측정했다.

🧪 Evaluation

학습 처리량은 AR이 50,620 tok/s, MDLM이 48,343 tok/s로 MDLM이 4.7%만 느렸고(총 학습 시간 107.9분 vs 113.0분), 확산 학습의 연산 오버헤드가 미미함을 보였다. 수렴 패턴에서 AR은 스텝 14,000에서 최소 검증 손실 1.589에 도달 후 과적합(20,000 스텝에서 1.622로 상승)했고, MDLM은 20,000 스텝까지 단조 감소(3.412)하며 여전히 개선 중이었다. 생성 다양성에서 AR 샘플의 99.8%가 동일한 단어("Once")로 시작하고 고유 첫 5-gram 비율이 3.3%에 불과한 반면, MDLM은 고유 첫 단어 36.1%, 고유 첫 5-gram 93.4%, Distinct-2 0.297(AR: 0.275), Self-BLEU 0.334(AR: 0.341)를 달성하여 구조적 다양성에서 압도적 우위를 보였다.

💡 Key Intuition & Lesson

AR 모델의 "접두사 모드 붕괴(prefix mode collapse)" 현상, 즉 좌-우 순차 생성이 가장 확률 높은 접두사에 고착되어 출력 다양성이 극도로 제한되는 문제는 MDLM의 비순차적 생성(모든 위치가 마스크 상태에서 시작하여 신뢰도 순으로 드러남)으로 구조적으로 해결됨을 보여준다. 또한 MDLM의 랜덤 마스킹이 일종의 암묵적 데이터 증강 역할을 하여 과적합을 지연시키는 정규화 효과가 있으며, 이는 동일 데이터에서 더 오래 학습할 수 있는 실용적 이점을 제공한다. 결론적으로 AR과 MDLM은 경쟁 관계가 아닌 상호 보완적 패러다임이며, 유창성이 중요한 과제(번역, 요약, 코드 생성)에는 AR이, 다양성이 중요한 과제(창작, 데이터 증강, 브레인스토밍)에는 MDLM이 적합하다는 용도별 분화의 방향성을 제시한다.

### 3. AgenticPay: A Multi-Agent LLM Negotiation System for Buyer-Seller Transactions

**트랙**: Agents | **링크**: [arXiv](https://arxiv.org/abs/2602.06008)

제공된 정보를 바탕으로 분석하겠습니다.

📋 Problem Definition

LLM 기반 에이전트가 자율적으로 협상하고 거래를 수행할 것으로 기대되지만, 기존 벤치마크에는 여러 에이전트 간의 언어 매개 경제적 상호작용(language-mediated economic interaction)을 원칙적으로 평가할 수 있는 설정이 부재하다. 특히 단순한 숫자 입찰이 아닌 자연어를 통한 다자간 협상 능력을 체계적으로 측정할 프레임워크가 필요하며, 이 논문은 구매자-판매자 거래 시나리오에서 LLM 에이전트의 협상 역량을 평가하기 위한 벤치마크인 AgenticPay를 제안한다.

📚 Background / Related Works

이 연구는 크게 세 가지 선행 연구 흐름과 관련된다. 첫째, LLM 에이전트 벤치마크 분야에서 AgentBench, GAIA, WebArena 등이 다양한 에이전트 능력을 평가해왔으나 경제적 협상에 특화된 설정은 제공하지 않았다. 둘째, 협상 및 전략적 상호작용 연구로는 Deal or No Deal(Lewis et al., 2017)이 양자 협상의 초기 연구를 제시했고, SOTOPIA나 Avalon 같은 사회적 시뮬레이션 벤치마크가 다자간 상호작용을 다뤘지만 경제적 합리성이나 시장 구조를 명시적으로 모델링하지는 않았다. 셋째, 게임이론 기반 LLM 평가 연구에서 최후통첩 게임이나 죄수의 딜레마 등을 활용한 연구들이 있었으나, 이들은 제한된 액션 공간과 단순한 페이오프 구조에 머물러 있어 실제 시장에서의 복잡한 다자간 언어 협상을 반영하기 어려웠다.

🔬 Main Methodology

AgenticPay는 구매자와 판매자가 각각 비공개 제약조건(private constraints)과 상품별 가치평가(product-dependent valuations)를 보유한 시장을 모델링하며, 합의에 도달하기 위해 다회차 자연어 협상을 수행하는 시뮬레이션 프레임워크이다. 프레임워크는 양자 간 흥정(bilateral bargaining)부터 다대다 시장(many-to-many markets)까지 110개 이상의 다양한 태스크를 지원하며, 시장 규모와 복잡도에 따라 체계적으로 난이도가 구성된다. 에이전트의 행동은 구조화된 액션 추출(structured action extraction) 방식으로 파싱되어, 자연어 발화 속에서 제안, 수락, 거절 등의 협상 행위를 식별한다. 평가 지표로는 실행 가능성(feasibility, 제약조건 충족 여부), 효율성(efficiency, 파레토 최적에 대한 근접도), 복지(welfare, 전체 사회적 잉여)의 세 축을 사용하여 협상 결과를 다차원적으로 측정한다. 이러한 설계는 단순한 가격 최적화를 넘어서 LLM이 장기적 전략적 추론(long-horizon strategic reasoning)을 수행할 수 있는지를 검증하는 데 초점을 맞추고 있다.

🧪 Evaluation

최신 프로프라이어터리 모델(GPT-4 계열, Claude 등)과 오픈웨이트 모델(Llama 계열 등)을 포함한 다양한 LLM을 110개 이상의 태스크에 걸쳐 벤치마킹하였으며, 모든 모델에서 협상 성능에 상당한 격차가 관찰되었다. 특히 시장 참여자 수가 증가하고 협상 라운드가 길어지는 다대다 시장 환경에서 성능 저하가 두드러졌는데, 이는 LLM이 장기적 전략 수립과 다자간 이해관계 조율에서 여전히 취약함을 보여준다. 프로프라이어터리 모델이 오픈웨이트 모델 대비 전반적으로 높은 feasibility와 welfare 점수를 기록했으나, 파레토 효율성 측면에서는 모든 모델이 최적 해에 크게 미치지 못하는 결과를 보였다.

💡 Key Intuition & Lesson

핵심 인사이트는 현재 LLM들이 단순 대화 능력은 뛰어나지만, 비공개 정보를 전략적으로 활용하고 다회차에 걸쳐 일관된 협상 전략을 유지하는 경제적 합리성(economic rationality)은 아직 크게 부족하다는 점이다. 이는 LLM 에이전트가 실제 상거래나 자동화된 거래 시스템에 배치되기 전에, 전략적 추론 능력과 제약조건 하에서의 의사결정 능력이 대폭 개선되어야 함을 시사한다. 또한 언어 기반 협상이라는 새로운 평가 축이 기존의 단순 QA나 코드 생성 벤치마크로는 포착할 수 없는 에이전트 능력의 중요한 차원을 드러낸다는 점에서, 향후 에이전트 평가 체계의 확장 방향을 제시한다.
