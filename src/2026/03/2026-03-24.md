### 1. Omni-WorldBench: Towards a Comprehensive Interaction-Centric Evaluation for World Models

**트랙**: LLM Post-training | **링크**: [arXiv](https://arxiv.org/abs/2603.22212)

📋 Problem Definition

기존 월드 모델(world model) 평가 벤치마크는 비디오 생성 모델의 시각적 품질이나 텍스트-비디오 정합성만 측정하거나, 3D 재구성 모델의 정적 메트릭에 의존하여 시간적 역학(temporal dynamics)을 근본적으로 무시하고 있다. 이 논문은 월드 모델링의 미래가 공간 구조와 시간 변화를 동시에 모델링하는 4D 생성(4D generation)에 있다고 주장하며, 핵심 역량인 "상호작용 반응(interactive response)", 즉 사용자의 행동이 시공간에 걸쳐 상태 전이를 어떻게 유발하는지를 체계적으로 평가하는 벤치마크가 전무하다는 문제를 제기한다.

📚 Background / Related Works

비디오 기반 월드 모델은 크게 비디오 생성 패러다임(Sora, CogVideo, Open-Sora 등)과 3D 재구성 패러다임(NeRF, 3D Gaussian Splatting 기반 모델들)의 두 축으로 발전해왔다. 기존 평가 벤치마크로는 비디오 생성 품질을 측정하는 VBench, EvalCrafter 등이 있고, 3D 재구성 측면에서는 PSNR/SSIM/LPIPS 같은 정적 렌더링 메트릭이 사용되어 왔으나, 이들 모두 상호작용에 의한 인과적 상태 변화를 평가하지 못한다. GameBench나 WorldSimBench처럼 게임/시뮬레이션 환경에서의 평가 시도가 있었으나, 다양한 상호작용 수준과 장면 유형을 포괄하는 체계적 벤치마크는 존재하지 않았다.

🔬 Main Methodology

Omni-WorldBench는 두 가지 핵심 구성요소로 이루어진다. 첫째, Omni-WorldSuite는 다양한 상호작용 수준(interaction level)과 장면 유형(scene type)을 아우르는 체계적 프롬프트 모음으로, 저수준(물체 조작 등)부터 고수준(복잡한 시나리오 기반 상호작용)까지의 행동을 포괄한다. 둘째, Omni-Metrics는 에이전트 기반(agent-based) 평가 프레임워크로, 상호작용 행동이 최종 결과(final outcome)뿐만 아니라 중간 상태 진화 궤적(intermediate state evolution trajectory)에 미치는 인과적 영향(causal impact)을 정량화한다. 이는 단순히 결과 프레임만 평가하는 것이 아니라, 행동으로 인한 시공간적 상태 전이 과정 전체를 추적한다는 점에서 기존 메트릭과 차별화된다. 평가 프레임워크가 에이전트 기반이라는 것은 LLM/VLM 에이전트를 활용하여 생성된 비디오나 4D 출력에서 상호작용의 인과적 결과를 자동으로 판단하는 구조를 의미한다.

🧪 Evaluation

비디오 생성 모델과 3D 재구성 모델을 포함한 18개의 대표적 월드 모델을 다양한 패러다임에 걸쳐 평가했다. 분석 결과, 현재의 월드 모델들은 상호작용 반응 능력에서 심각한 한계를 드러냈으며, 특히 행동에 따른 인과적 상태 전이를 정확하게 반영하는 데 실패하는 경우가 빈번했다. 구체적 수치는 논문 전문에서 확인이 필요하나, 저자들은 비디오 생성 모델과 3D 재구성 모델 모두 시각적 품질에서는 양호하더라도 상호작용 기반 평가에서는 현저히 낮은 성능을 보인다는 점을 강조하고 있다.

💡 Key Intuition & Lesson

이 논문의 핵심 인사이트는 월드 모델의 진정한 능력은 "얼마나 사실적인 영상을 만드는가"가 아니라 "사용자의 행동에 대해 물리적으로 타당한 상태 변화를 얼마나 정확히 예측하는가"에 있다는 것이다. 기존 평가 체계가 시각적 충실도에 편중되어 있어 상호작용이라는 본질적 차원을 놓치고 있었으며, 4D 세계 모델링의 발전을 위해서는 최종 결과뿐 아니라 중간 상태 궤적까지 인과적으로 추적하는 평가 패러다임으로의 전환이 필수적이다. 이는 향후 로보틱스, 자율주행, 게임 시뮬레이션 등 인터랙티브 응용에서 월드 모델을 배포하기 전 반드시 검증해야 할 기준을 제시한다.

### 2. Manifold-Aware Exploration for Reinforcement Learning in Video Generation

**트랙**: LLM Post-training | **링크**: [arXiv](https://arxiv.org/abs/2603.21872)

📋 Problem Definition

비디오 생성 모델에 GRPO(Group Relative Policy Optimization)를 적용할 때, ODE 샘플러를 SDE 샘플러로 변환하는 과정에서 1차 근사(first-order approximation)에 의한 과도한 노이즈가 주입되어 롤아웃 품질이 저하되고 보상 추정이 불안정해지는 문제를 다룬다. 비디오 생성의 해 공간(solution space)은 언어 모델이나 이미지에 비해 훨씬 복잡하고 구조화되어 있어, 탐색(exploration) 과정에서 데이터 매니폴드를 벗어나면 시간적 떨림(temporal jitter)과 같은 아티팩트가 발생하고 학습이 불안정해진다. 이 논문의 핵심 질문은 "사전학습된 모델이 정의하는 유효한 비디오 데이터 매니폴드 근방에서 탐색을 어떻게 제약할 것인가"이다.

📚 Background / Related Works

DDPO(Black et al., 2023)와 DPOK(Fan et al., 2023)가 디노이징 과정을 MDP로 모델링하여 정책 경사 추정을 가능하게 한 초기 연구이며, 이후 언어 모델의 GRPO(DeepSeek-R1의 Shao et al., 2024; Guo et al., 2025)에서 영감을 받아 FlowGRPO(Liu et al., 2025b)와 DanceGRPO(Xue et al., 2025)가 ODE-to-SDE 변환을 통해 시각 생성에 GRPO를 적용했다. 그러나 이 방법들은 Euler 스타일 이산화와 1차 근사에 의존하여 SDE 노이즈 표준편차를 도출하기 때문에, 고노이즈 구간에서 과도한 에너지가 주입되어 매니폴드 이탈 문제가 발생한다. 또한 기존 정렬 프레임워크들은 고정 KL 제약(안정성은 높지만 가소성 부족)과 완화된 제약(보상 해킹 또는 catastrophic forgetting 유발) 사이의 안정성-가소성 딜레마에 직면해 있었다.

🔬 Main Methodology

SAGE-GRPO는 미시(micro) 수준과 거시(macro) 수준의 두 축으로 매니폴드 인식 탐색을 구성한다. 미시 수준에서는, 기존 방법들이 면적 기반 1차 근사로 노이즈 분산을 계산하는 것과 달리, 확산 계수 ε_t = η√(σ_t/(1-σ_t))를 구간 [σ_{t+1}, σ_t]에서 적분하여 로그 곡률 보정항 log((1-σ_{t+1})/(1-σ_t))을 포함한 정밀한 분산 Σ_t를 도출한다. 이 로그항은 신호 계수 (1-σ_t)의 기하학적 수축을 반영하며, 선형 근사가 포착하지 못하는 부분을 보정하여 더 작고 매니폴드에 정렬된 탐색 영역을 생성한다. 또한 타임스텝 간 그래디언트 노름이 수 자릿수(order of magnitude) 차이가 나는 불균형 문제(낮은 노이즈에서 폭발, 높은 노이즈에서 소실)를 해결하기 위해, 중앙값(median) 기반 정규화 S_t = Median({N_τ})/(N_t + ε)를 적용하는 Gradient Norm Equalizer를 도입한다. 거시 수준에서는, 고정 KL(위치 고정으로 인한 과소적합)과 스텝별 KL(속도만 제어하여 누적 드리프트 미제한)의 한계를 동시에 극복하기 위해, N 스텝마다 참조 정책을 갱신하는 Periodical Moving Anchor(위치 제어)와 이전 스텝 대비 KL(속도 제어)을 결합한 Dual Trust Region 목적함수 L_KL = β_pos · D_KL(π_θ||π_{ref_N}) + β_vel · D_KL(π_θ||π_{k-1})를 제안한다.

🧪 Evaluation

HunyuanVideo 1.5 모델에 VideoAlign 보상 모델(별도의 보상 모델 파인튜닝 없이 원본 사용)을 적용하여 평가했으며, Setting A(균등 보상 가중치 w_vq=w_mq=w_ta=1.0)와 Setting B(정렬 중심 가중치 w_vq=0.5, w_mq=0.5, w_ta=1.0) 두 가지 설정에서 DanceGRPO, FlowGRPO, CPS와 비교했다. Setting B의 Dual Moving KL 구성에서 SAGE-GRPO는 Overall 0.8066, VQ -0.4765, MQ -0.2384, CLIPScore 1.5216, PickScore 0.5484를 달성하여 모든 베이스라인을 압도적으로 상회했으며, 이는 두 번째로 높은 CPS(w/o KL)의 Overall 0.3694 대비 두 배 이상의 성능이다. KL 가중치 스케줄링 실험에서는 10^{-7}에서 10^{-5}로의 2단계 스케줄이 VQ, MQ, TA 세 지표 모두에서 가장 강하고 일관된 개선을 보여, 점진적으로 trust region을 조여가는 전략의 유효성을 확인했다.

💡 Key Intuition & Lesson

이 논문의 핵심 통찰은 비디오 GRPO에서의 불안정성이 "탐색 자체의 문제"가 아니라 "탐색이 데이터 매니폴드를 벗어나는 문제"라는 재정의에 있으며, ODE-to-SDE 변환 시 1차 근사의 절단 오차가 고노이즈 구간에서 매니폴드 이탈을 유발한다는 구체적 원인을 밝힌 점이 중요하다. 제어 이론의 위치-속도 제어기(position-velocity controller) 관점에서 KL 제약을 재해석하여, 주기적 앵커 갱신(위치 제어)과 스텝별 제약(속도 제어)의 이중 구조가 안정성과 가소성을 동시에 확보할 수 있다는 설계 원칙은 비디오 생성뿐 아니라 복잡한 구조적 출력을 가진 다른 생성 모델의 RL 정렬에도 일반적으로 적용 가능한 교훈을 제공한다.

### 3. RuleSmith: Multi-Agent LLMs for Automated Game Balancing

**트랙**: Agents | **링크**: [arXiv](https://arxiv.org/abs/2602.06232)

Semantic Scholar 도구 접근도 허용되지 않았습니다. 제공해주신 abstract 정보를 바탕으로 분석하겠습니다.

📋 Problem Definition

게임 밸런싱은 반복적인 플레이테스팅, 전문가의 직관, 그리고 방대한 수동 튜닝을 요구하는 오래된 난제이다. 특히 문명(Civilization) 류의 복잡한 전략 게임에서는 이질적인 팩션, 경제 시스템, 생산 규칙, 전투 메커닉 등 수십 개의 튜닝 가능한 파라미터가 존재하여, 이들 간의 상호작용을 사람이 수작업으로 균형 잡는 것은 비용이 극히 크고 확장 가능하지 않다. RuleSmith는 이 과정을 LLM 기반 멀티 에이전트 시뮬레이션과 베이지안 최적화를 결합하여 자동화하는 최초의 프레임워크를 제안한다.

📚 Background / Related Works

게임 밸런싱 자동화 연구로는 강화학습(RL) 기반 접근이 대표적이며, OpenAI Five(Dota 2)나 AlphaStar(StarCraft II) 등이 게임 내 에이전트 학습에 활용되었으나, 이들은 밸런싱이 아닌 최적 플레이 전략 학습에 초점을 맞추었다. LLM을 게임 에이전트로 활용한 연구로는 Generative Agents(Park et al., 2023)나 Voyager(Wang et al., 2023) 등이 있으며, 텍스트 기반 환경에서 LLM의 의사결정 능력을 보여주었다. 베이지안 최적화는 블랙박스 함수의 효율적 탐색에 널리 쓰이는 기법으로, 게임 파라미터 튜닝과 같은 고비용 평가 환경에 적합하다.

🔬 Main Methodology

RuleSmith는 세 가지 핵심 구성요소를 결합한다: (1) 게임 엔진(CivMini), (2) 멀티 에이전트 LLM 셀프플레이, (3) 다차원 규칙 공간에서의 베이지안 최적화. CivMini는 문명 스타일 게임을 단순화한 테스트베드로, 이질적 팩션, 경제, 생산, 전투 등의 메커닉을 튜닝 가능한 파라미터로 관리한다. LLM 에이전트들은 텍스트로 된 룰북과 게임 상태를 해석하여 행동을 생성하며, 셀프플레이를 통해 승률 격차(win-rate disparity) 같은 밸런스 지표를 빠르게 평가한다. 파라미터 탐색의 효율성을 위해, acquisition function 기반 적응적 샘플링과 이산 투영(discrete projection)을 통합한 베이지안 최적화를 사용한다. 유망한 파라미터 후보에는 더 많은 평가 게임을 할당하여 정확한 평가를 수행하고, 탐색적 후보에는 적은 게임을 할당하여 효율적 탐색을 달성하는 적응적 자원 배분 전략이 핵심 기여이다.

🧪 Evaluation

CivMini 환경에서 실험을 수행했으며, 밸런스 지표로는 팩션 간 승률 격차(win-rate disparity)를 사용했다. RuleSmith는 반복적인 베이지안 최적화 루프를 통해 높은 수준의 균형 잡힌 게임 설정으로 수렴하는 것을 보여주었으며, 최적화 과정에서 도출된 규칙 조정값은 해석 가능(interpretable)하여 하위 게임 시스템에 직접 적용 가능한 형태로 제공된다. 구체적으로, 적응적 샘플링이 균일 샘플링 대비 더 적은 평가 횟수로 동등하거나 우수한 밸런스 설정을 찾아내는 효율성을 입증했다.

💡 Key Intuition & Lesson

이 논문의 핵심 인사이트는 LLM이 단순히 텍스트 생성 도구가 아니라, 복잡한 규칙 체계를 이해하고 전략적 의사결정을 수행할 수 있는 "시뮬레이션 서로게이트(surrogate)"로 기능할 수 있다는 점이다. 기존에는 게임 밸런싱을 위해 사람 플레이어나 정교한 RL 에이전트가 필요했지만, LLM 셀프플레이가 이를 대체할 수 있음을 보여줌으로써, 게임뿐 아니라 정책 시뮬레이션, 경제 모델링 등 복잡한 멀티에이전트 환경의 설계 자동화로 확장 가능한 패러다임을 제시한다. 또한 베이지안 최적화의 적응적 샘플링을 통해 평가 비용이 높은 환경에서도 효율적 탐색이 가능함을 보여준 점은, LLM 기반 시뮬레이션의 실용성을 높이는 중요한 엔지니어링 교훈이다.
